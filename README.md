# Rider Operations MIS Dashboard | Excel Analytics Project

## Project Overview

This project demonstrates how Microsoft Excel can be used to build an end-to-end Management Information System (MIS) dashboard for monitoring rider operations in a delivery startup.

The solution transforms raw operational data into interactive dashboards, KPI reports, business insights, and management-ready summaries using Excel's analytical capabilities.

---

## Business Problem

Operations teams require a centralized reporting system to monitor rider productivity, attendance, earnings, deductions, and payouts. Manual reporting is time-consuming and prone to errors, making it difficult to identify performance trends and operational risks.

---

## Objective

Build a dynamic Excel-based MIS solution that:

* Tracks rider productivity
* Calculates rider payouts
* Measures attendance
* Monitors bonus eligibility
* Generates management dashboards
* Produces weekly operational reports

---

## Dataset

The project uses a simulated rider operations dataset representing a Delhi-based delivery startup.

**Dataset includes:**

* Rider ID
* Rider Name
* Date
* Trips Completed
* Earnings per Trip
* Bonus Eligibility
* Deductions
* Attendance
* Zone
